Make functions great again

Posted on May 19, 2017 by palutz

Tags: functional programming, community

After some time, I came back to provide a technical talk.

This time it was my first talk about functional programming , with a particular set up: first, an introductory part focused on the concepts behind the functional programming paradigm and then a fast exploration of F# , an excellent language, mature, OSS and functional first finally getting the traction it deserves (IMHO).

In this post, I will just share my slides for the first part, since the second one was almost all code, leveraging the very nice tools open source and community driven that we can use while working with F#: dotnet core, Visual Studio Code and Ionide. I will try to wrap up also the second part in a better presentation.

Meanwhile, I hope you will enjoy my introductory presentation
